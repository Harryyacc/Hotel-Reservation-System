# Hotel Reservation System

## Project Overview
A menu-driven hotel reservation system that allows users to book rooms, cancel reservations, view available rooms, and display all bookings. Uses CSV file for data storage.

## Features
- Add room booking with client details
- Cancel bookings with verification
- Display available rooms
- Show all reservations
- Menu-driven program
- File handling
